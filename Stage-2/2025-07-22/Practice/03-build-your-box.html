Box Model Practice 3: Build Your Own Box

Your Task: Style the challenge box to meet these requirements:

  • Content area: 180px wide and 90px tall
  • Padding: 20px on all sides
  • Border: 3px solid orange
  • Margin: 15px on all sides
  • Background color: lightyellow

Instructions:

  1. Look at the CSS section in the <head> of this file
  2. Find the .your-box selector
  3. Add CSS properties to match the requirements above
  4. Save the file and refresh to see your changes
  5. Right-click your box and inspect it to verify measurements

Questions:

  • Before coding, calculate: What will the total width and height be?
  • After coding, does your calculation match what dev tools shows?
  • What happens if you add box-sizing: border-box to your box?
  • How would you make your box exactly 250px total width using border-box?

Answers:

  • Total dimensions: Width: 180 + 40 + 6 = 226px, Height: 90 + 40 + 6 = 136px (margin doesn't count toward total size)
  • Calculation check: Your calculations should match dev tools measurements
  • With border-box: Total size becomes exactly 180x90px, content area shrinks
  • 250px with border-box: Set width: 250px, content becomes 250 - 40 - 6 = 204px

CSS Solution:

.your-box {
    width: 180px;
    height: 90px;
    padding: 20px;
    border: 3px solid orange;
    margin: 15px;
    background-color: lightyellow;
}

Experiments:

  1. Add box-sizing: border-box to your box and inspect it - how do the measurements change?
  2. Create a second challenge: make a box that's exactly 250px total width using border-box sizing
  3. Try different padding values on each side: padding: 10px 20px 15px 5px - what's the total width now?

Experiment Results:

  • Border-box effect: Total size becomes 180x90px, but content area shrinks to 134x44px
  • 250px challenge: width: 250px; box-sizing: border-box; gives exactly 250px total
  • Different padding: Total width = 180 + (20+5) + 6 = 211px (left+right padding = 25px total)

Key insight: Building boxes from scratch helps you understand how each property contributes to the final result!
