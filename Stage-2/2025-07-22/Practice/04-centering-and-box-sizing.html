Your Challenge:

Style me: 200px total, centered!

Box Model Practice 4: Centering and Box-Sizing

Instructions:

  1. Right-click each example box and inspect them
  2. Compare the centering: left-aligned vs centered boxes
  3. Compare the box-sizing: default vs border-box examples
  4. Notice how same CSS width produces different total sizes
  5. Style the challenge box to meet the requirements below

Questions:

  • What's the difference between margin: 15px 0 and margin: 15px auto?
  • Both sizing examples have width: 120px - why are their total widths different?
  • Which box-sizing approach gives you more predictable layouts?
  • How would you center a box that has margin: 10px 0?

Answers:

  • Margin difference: margin: 15px 0 = left-aligned, margin: 15px auto = centered horizontally
  • Width difference: Default = 120px + 40px + 6px = 166px total, Border-box = exactly 120px total
  • More predictable: Border-box sizing - you set the width you want and it stays that width
  • Centering fix: Change it to margin: 10px auto

Note: auto margins automatically distribute available space equally on left and right!

Challenge Requirements:

Style the challenge box to meet these requirements:

  • Total width: exactly 200px (including padding and border)
  • Padding: 15px on all sides
  • Border: 2px solid orange
  • Centered horizontally on the page
  • Background color: lightyellow

Experiments:

  1. Try changing the centered box's margin from auto to 0 - what happens?
  2. Add box-sizing: border-box to the default-sizing box - how does its total width change?
  3. Remove box-sizing: border-box from the border-box example - what changes?
  4. Style the challenge box to meet the requirements above

Experiment Results:

  • Changing auto to 0: Box moves to left side, no longer centered
  • Adding border-box to default: Total width becomes exactly 120px (shrinks from 166px)
  • Removing border-box: Total width becomes 166px (expands from 120px)
  • Challenge solution: width: 200px; box-sizing: border-box; margin: 15px auto; padding: 15px; border: 2px solid orange; background-color: lightyellow;

Key insight: margin: auto + box-sizing: border-box = predictable centering and sizing!
