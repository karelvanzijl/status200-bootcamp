Box Model Practice 1: Basic Box Inspection

Instructions:

  1. Right-click the blue box above and select "Inspect" (or "Inspect Element")
  2. Look for the box model diagram in the dev tools (usually at the bottom of the Elements panel)
  3. Identify each part: content (blue), padding (green), border (yellow), margin (orange)
  4. Note the measurements for each section

Questions:

  • What is the content width and height?
  • How much padding is applied on each side?
  • What is the border thickness?
  • How much margin space is around the box?
  • What is the total width of the box including padding and border?

Answers:

  • Content width and height: 200px × 100px
  • Padding on each side: 20px
  • Border thickness: 3px
  • Margin space around box: 30px
  • Total box width: 200px (content) + 40px (padding) + 6px (border) = 246px

Note: Margin is NOT included in the box width - it just creates space around the box!

Experiments:

  1. In the CSS above, try changing padding: 20px to padding: 10px - what happens to the box size?
  2. Try changing margin: 30px to margin: 50px - what happens to the space around the box?
  3. Try changing border: 3px solid black to border: 10px solid red - how does this affect the total size?

What happens when you make the changes:

  • Changing padding to 10px: Box gets smaller (total width becomes 226px)
  • Changing margin to 50px: More space around box, but box size stays 246px
  • Changing border to 10px: Box gets bigger (total width becomes 260px)
